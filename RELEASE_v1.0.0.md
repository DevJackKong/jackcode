# JackCode v1.0.0

Initial release of JackCode.

## Highlights

- Qwen 3.6 as the primary development model
- GPT-5.4 as the audit / verifier model
- DeepSeek removed from the main architecture
- 100% passing test suite
- CLI supports truthful dry-run and approved execution flow
- npm package is prepared for publishing

- Two-model architecture focused on cost and quality
- Repo scanning and impact analysis foundations
- Patch engine with apply / verify / rollback flow
- CLI chat and execute workflows
- Documentation, demos, and examples
